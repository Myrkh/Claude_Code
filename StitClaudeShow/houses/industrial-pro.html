03 RULES · trap / exit POLICY-REV.02

The density can become illegible if the discipline breaks. These are the two rules that separate a Bloomberg terminal from a "dashboard template."

ALLOW

Use tabular numerics on every column of data. font-variant-numeric: tabular-nums is not optional.

Signal-coded colors carry semantic meaning — green is success, red is critical, amber is system. Respect the convention.

DENY

Decorative gradients on data tables. If the data is good, it does not need help. If it isn't, a gradient won't save it.

Fonts below 12px, ever. Dense does not mean small — dense means information-rich.

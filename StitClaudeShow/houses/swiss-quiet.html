02 · Motion

How this house moves.

Short, standard easing. No spring. Swiss design respects the grid; motion respects the clock. 200ms for everything. No animation if a static state communicates the same thing.

03 · The trap

Do one thing, refuse another.

Every house has a characteristic failure mode. These are the traps to avoid.

Do

Align everything to the grid. 8pt grid if you must choose. Every edge aligned is a vote for coherence.

Pick one color and commit. Swiss-Quiet is monochrome plus one, not palette exploration.

Don't

Break the grid for creative effect. If the grid is right, it doesn't need breaking.

Use drop shadows to create hierarchy. Use weight, size, and position. Shadows are a crutch.

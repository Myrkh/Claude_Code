02 Motion max 120ms · no spring

03 Rules the two that matter
Do

Display keyboard shortcuts on every hoverable element. ⌘K, ⌘↵, ⌘\.

Build a global command palette. It's the one feature that separates pro tools from consumer apps.

Don't

Onboard with a tour. Power users skip tours. Provide searchable help reachable from anywhere instead.

Hide features behind illustrations. Every pixel in Power-Dense is either information or affordance.

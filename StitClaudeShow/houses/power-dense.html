02 · Motion

How this house moves.

Minimal motion. Power users move faster than animations. 120ms max for feedback. No spring on navigation — it feels like lag when you know where you're going. Animation must never block input.

✓--duration-fast: 120ms  ·  the default
✓--ease-standard  ·  everything
✕no spring  ·  spring reads as lag to power users
✓keyboard-first  ·  every action needs a shortcut
03 · The trap

Do one thing, refuse another.

Every house has a characteristic failure mode. These are the traps to avoid.

Do

Display keyboard shortcuts on every hoverable element. ⌘K, ⌘↵, ⌘\.

Build a global command palette. It's the one feature that separates pro tools from consumer apps.

Don't

Onboard with a tour. Power users skip tours. Provide help reachable from anywhere instead.

Hide features behind cute illustrations. Every pixel in Power-Dense is information or affordance.

Master_Claude

Portfolio of skills built for Claude — each one a self-contained Markdown folder loaded by the Claude runtime to give the model a specific discipline.

Skills
  • stitclaude/ — frontend design discipline · 8 houses · OKLCH tokens
  • prism/ — functional safety · SIL / SIF · IEC 61508
  • kore/ — engineering document intelligence
  • doctrack/ — multi-disciplinary doc tracking
  • infitex/ — infinite-canvas LaTeX writing
  • deusex/ — AI-arbited RPG sandbox
Install a single skill

Clone the repo, then symlink the skill folder into Claude's skills directory — see the INSTALL.md inside each skill for specifics.

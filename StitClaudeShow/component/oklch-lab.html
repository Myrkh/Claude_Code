<script>
(function() {
  const $ = id => document.getElementById('stit-oklch-' + id);
  const sL = $('l'), sC = $('c'), sH = $('h');
  const vL = $('val-l'), vC = $('val-c'), vH = $('val-h');
  const preview = $('preview'), formula = $('formula'), palette = $('palette');

  for (let i = 0; i < 12; i++) {
    const d = document.createElement('div');
    d.className = 'stit-oklch__step';
    palette.appendChild(d);
  }

  const LSteps = [0.99, 0.975, 0.95, 0.91, 0.86, 0.78, 0.68, 0.56, 0.45, 0.38, 0.28, 0.18];
  const CSteps = [0.008, 0.02, 0.045, 0.07, 0.09, 0.11, 0.13, 0.15, 0.16, 0.15, 0.13, 0.08];

  function update() {
    const L = parseInt(sL.value, 10);
    const C = parseInt(sC.value, 10);
    const H = parseInt(sH.value, 10);
    const Lf = (L / 100).toFixed(2);
    const Cf = (C / 100).toFixed(2);
    vL.textContent = Lf;
    vC.textContent = Cf;
    vH.textContent = H;
    const color = `oklch(${Lf} ${Cf} ${H})`;
    preview.style.setProperty('--color-preview', color);
    formula.textContent = color;
    palette.querySelectorAll('.stit-oklch__step').forEach((step, i) => {
      const Cstep = (CSteps[i] * (C / 20)).toFixed(3);
      step.style.background = `oklch(${LSteps[i].toFixed(3)} ${Cstep} ${H})`;
    });
  }
  [sL, sC, sH].forEach(s => s.addEventListener('input', update));
  update();
})();
</script>

Swiss-Quiet

Linear's entire palette was born this way — one core hue, chroma dropped slightly for dark mode, perceptually even steps.

Industrial-Pro

Scientific tools need contrast that works for red/green colorblind users. OKLCH makes safe derivation of signal colors trivial.

Editorial

Warm cream and one deep accent — the only way to get that effect without muddy transitions is OKLCH.

Apple-Craft

iOS 26 Liquid Glass uses OKLCH for its tint adaptation. Apple's own System Colors ship in P3 gamut, which OKLCH addresses natively.

  • ✕
    Deriving dark-mode colors by inverting lightness only

    Flipping oklch(0.95 0.01 260) to oklch(0.05 0.01 260) gives a flat, dead dark. Accents that worked on cream will vibrate on deep near-black.

    Instead: drop chroma by 10–20 % when switching to dark, and raise lightness slightly on the accent so it still pops against the new surface.

  • ✕
    Picking hues “that feel vibrant together”

    Coral + teal + indigo chosen by eye on a Figma canvas. Each has different chroma and lightness, so they fight for attention.

    Instead: keep one accent at full chroma, everything else at the same L with chroma dropped. The palette settles instantly.

  • ✕
    Hardcoded #RRGGBB literals in components

    background: #5E6AD2 sprinkled across components. When brand color changes, you grep-and-replace and miss three instances.

    Instead: commit a single oklch() value as a CSS custom property, reference it everywhere. Change one token, everything updates.

  • ✕
    Ignoring the APCA contrast check

    WCAG 2 says #777 on white passes AA at 14pt. APCA says it barely clears Lc 60 — unreadable for body text.

    Instead: use an APCA calculator. Target Lc 75 for body, Lc 60 for large headings. It's closer to how humans actually see.
